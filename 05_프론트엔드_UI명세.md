## 📄 템플릿 구조

### 1. index.html (입력 폼)

**구조**:
```html
<body>
    <div class="container">
        <div class="glass-card">
            <!-- 로고 -->
            <i class="fa-solid fa-moon text-gradient"></i>
            <h1 class="text-gradient">GLOW글로우:갓생사주</h1>
            <p>운명을 읽고 나를 켜다</p>
            
            <!-- 폼 -->
            <form action="/loading" method="POST">
                <input name="name" placeholder="홍길동">
                <input type="radio" name="gender" value="male"> 남성
                <input type="radio" name="gender" value="female"> 여성
                <input type="date" name="birth_date">
                <input type="time" name="birth_time">
                <button class="btn-primary">운세 확인하기</button>
            </form>
        </div>
    </div>
</body>
```

**폼 필드**:
| 필드명 | 타입 | 필수 | 설명 |
|--------|------|------|------|
| name | text | ✓ | 이름 |
| gender | radio | ✓ | male/female |
| birth_date | date | ✓ | 양력 생년월일 |
| birth_time | time | ✓ | 태어난 시간 |

## 🔀 탭 전환 JavaScript

```javascript
function openTab(evt, tabName) {
    // 모든 탭 컨텐츠 숨김
    var contents = document.querySelectorAll('.tab-content');
    contents.forEach(function (content) {
        content.style.display = 'none';
        content.classList.remove('active');
    });

    // 모든 탭 버튼 비활성화
    var buttons = document.querySelectorAll('.tab-btn');
    buttons.forEach(function (btn) {
        btn.classList.remove('active');
    });

    // 선택된 탭 활성화
    document.getElementById(tabName).style.display = 'block';
    document.getElementById(tabName).classList.add('active');
    evt.currentTarget.classList.add('active');
}
```
