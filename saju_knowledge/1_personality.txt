# [지식 주입용-1] 사주 명식, 성격, 격국, 강약, 용신, 공망 비법 (코어 지식)

이 문서는 AI가 사용자의 사주 뼈대를 해석하기 위해 참고하는 대표님의 비급 코어 로직입니다.

### 1. 지장간 (地藏干) 
- 子: 壬(10) / 癸(20) - 순수한 물
- 丑: 癸(9) / 辛(3) / 己(18) - 차가운 겨울 흙
- 寅: 戊(7) / 丙(7) / 甲(16) - 만물이 깨어나는 봄
- 卯: 甲(10) / 乙(20) - 순수한 나무
- 辰: 乙(9) / 癸(3) / 戊(18) - 물을 머금은 봄의 흙
- 巳: 戊(7) / 庚(7) / 丙(16) - 초여름 불
- 午: 丙(10) / 己(9) / 丁(11) - 뜨거운 불
- 未: 丁(9) / 乙(3) / 己(18) - 뜨거운 여름 흙
- 申: 戊(7) / 壬(7) / 庚(16) - 금과 수의 태동
- 酉: 庚(10) / 辛(20) - 순수한 쇠
- 戌: 辛(9) / 丁(3) / 戊(18) - 가을 흙
- 亥: 戊(7) / 甲(7) / 壬(16) - 큰 물

### 2. 천간합과 지지충
- [천간합]: 갑기합화 토(중정지합), 을경합화 금(인의지합), 병신합화 수(위엄지합), 정임합화 목(인수지합), 무계합화 화(무정지합)
- [지지충]: 자오충(감정,이동), 축미충(부동산,형제), 인신충(교통,개척), 묘유충(대인관계,이별), 진술충(투쟁,직업변동), 사해충(해외이동,정신변화)

### 3. 신살 (특수 기운)
- [삼합 기준]: 해/묘/미생은 도화(子),역마(巳),화개(未). 인/오/술생은 도화(卯),역마(申),화개(戌) 등. 도화(매력), 역마(이동), 화개(명예/고독).
- [기둥 기준]: 백호대살(폭발력, 프로 기질), 괴강살(리더십, 카리스마)
- [귀문관살]: 자유, 축오, 인미, 묘신, 진해, 사술. (직관력, 예술성, 예민함)
- [천을귀인]: 일간 기준 나를 돕는 최고의 길신 수호천사 기운.

### 4. 십이운성 (에너지 스펙트럼)
- 절, 태, 양, 장생, 목욕, 관대, 건록, 제왕, 쇠, 병, 사, 묘의 주기.
- 장생, 건록, 제왕은 강한 에너지. 쇠, 병, 사, 묘, 절은 정신적, 수용적 에너지.

### 5. 격국 (내 그릇의 크기)
- 월지 지장간 중 천간에 투출된 글자(정기->중기->초기 순)를 기준으로 격을 잡는다.
- 투출되지 않으면 월지 정기 자체가 격이 된다.
- 건록격(스스로 일어섬), 양인격(프로페셔널, 치열함 - 월지가 겁재일 때)의 특수격 존재.

### 6. 강약 (신강/신약 판별)
- 내 편(비견, 겁재, 정인, 편인) 점수 계산: 월지(30점), 일지/월간/시간(각 15점), 년지/시지(각 10점), 년간(5점).
- 합계 50점 이상 '신강': 주관이 확실, 리더십, 재물/관성을 취할 뚝심이 있음.
- 합계 50점 미만 '신약': 상황 대처, 협력, 조직에 속했을 때 유리, 유연함.

### 7. 용신과 기신 (사주의 병과 약)
- 조후 용신: 여름생(사/오/미 월)은 너무 더우니 水가 용신. 겨울생(해/자/축 월)은 너무 추우니 火가 용신.
- 억부 용신: 신강한 자는 식/재/관으로 힘을 부드럽게 설기/통제. 신약한 자는 인/비로 힘을 보충해 주어야 함.

### 8. 공망 (채워지지 않는 결핍)
- 그룹 짝을 짓고 남는 2개의 지지를 의미.
- 재성 공망: 돈에 대한 갈증. 관성 공망: 조직의 답답함, 내 사업 선호. 인성 공망: 인정과 사랑에 대한 목마름. 식상 공망: 표현의 한계, 조용한 내실 유리.
